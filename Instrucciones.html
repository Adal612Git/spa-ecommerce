Instalación y Pruebas

  1. Clona el repositorio: git clone https://github.com/tu-org/spa-ecommerce.git
  2. Instala dependencias en la raíz: npm install
  3. En una terminal, inicia el backend: npm run dev -w backend (puerto 3000)
  4. En otra terminal, inicia el frontend: npm run dev -w frontend (puerto 9000)
  5. Abre el navegador en http://localhost:9000 y sigue el flujo de compra completo.
  6. Para ejecutar las pruebas automáticas: npm test
